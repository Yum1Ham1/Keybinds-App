## Descrição / Description

**Português:**  
O **Keybinds-App** é um sistema de gerenciamento de keybinds de texto para Windows.  
Ele permite que você associe qualquer tecla a um texto específico, que será digitado automaticamente quando a tecla for pressionada.  

Principais funcionalidades:
- Captura de teclas de forma prática;
- Listener configurável para ligar/desligar rapidamente;
- Adição, edição e remoção de keybinds;
- Salvar e carregar keybinds localmente;
- Suporte a modo claro/escuro com transição suave;
- Interface moderna em preto e azul com efeitos de destaque;
- Footer com créditos e links para GitHub e LinkedIn.

**English:**  
**Keybinds-App** is a text keybind management system for Windows.  
It allows you to assign any key to a specific text that will be automatically typed when the key is pressed.  

Main features:
- Easy key capture;
- Configurable listener to turn keybinds on/off;
- Add, edit, and delete keybinds;
- Save and load keybinds locally;
- Support for light/dark mode with smooth transition;
- Modern black & blue interface with highlight effects;
- Footer with credits and links to GitHub and LinkedIn.

---

## Funcionalidades / Features

**Português:**  
1. **Capturar Tecla** – Clique para capturar uma tecla do teclado e associar a um texto.  
2. **Adicionar Keybind** – Associe o texto digitado à tecla capturada.  
3. **Editar / Deletar Keybinds** – Faça alterações rápidas diretamente na lista de keybinds.  
4. **Listener** – Ligue ou desligue o listener global para ativar os keybinds.  
   - Tecla padrão: **PAUSE**, mas você pode alterar para qualquer tecla.  
5. **Tema** – Alternância entre modo claro e escuro com cores e efeitos sofisticados.  
6. **Salvamento Local** – Todos os keybinds são salvos automaticamente em `keybinds_data/keybinds.json`.  
7. **Footer de Créditos** – Mostra o autor e links para GitHub e LinkedIn.

**English:**  
1. **Capture Key** – Click to capture a key from your keyboard and assign it to a text.  
2. **Add Keybind** – Assign the typed text to the captured key.  
3. **Edit / Delete Keybinds** – Make quick edits directly in the keybinds list.  
4. **Listener** – Turn the global listener on/off to activate keybinds.  
   - Default key: **PAUSE**, but it can be changed to any key.  
5. **Theme** – Switch between light and dark mode with smooth transitions.  
6. **Local Save** – All keybinds are automatically saved in `keybinds_data/keybinds.json`.  
7. **Footer Credits** – Shows the author and links to GitHub and LinkedIn.
